# Nanny.Services

Застосунок для компанії, що надає послуги нянь. Користувач може переглядати нянь,
сортувати та фільтрувати їх, додавати в «обране», відкривати детальну інформацію з
відгуками та надсилати заявку на зустріч. Реалізовано авторизацію та приватну
сторінку «Favorites».

## Технології

- **Next.js 16** (App Router) + **React 19** + **TypeScript**
- **Firebase** — Authentication (email/password) та Realtime Database
- **@tanstack/react-query** — отримання та кешування даних
- **zustand** — глобальний стан (користувач, «обране», фільтр, модалки)
- **react-hook-form** + **yup** — форми та валідація
- **react-hot-toast** — сповіщення
- **CSS Modules** — стилізація, гумова верстка 320–1440px

## Функціонал (за ТЗ)

- Сторінка **Home** із заголовком, слоганом та кнопкою переходу до «Nannies».
- Сторінка **Nannies**: сортування A→Z / Z→A, фільтр за ціною (менше/більше 10$),
  сортування за популярністю (рейтинг за зростанням/спаданням), «Show all».
- Рендер по **3 картки**, решта — по кнопці **Load more**.
- Кнопка **«серце»**: неавторизованому показує сповіщення й пропонує вхід;
  авторизованому — додає/видаляє няню в «обране»,
  стан зберігається після оновлення сторінки.
- **Read more** розкриває картку з відгуками батьків.
- **Make an appointment** — модальне вікно з формою (react-hook-form + yup).
- Приватна сторінка **Favorites** зі стилізацією, аналогічною «Nannies».
- Усі модалки закриваються по «хрестику», кліку на backdrop та клавіші `Esc`.
